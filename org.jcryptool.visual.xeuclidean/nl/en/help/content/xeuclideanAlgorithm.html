1 The Extended Euclidean Algorithm

The Extended Euclidean Algorithm computes the greatest common divisor of two integers p and q. In addition, the algorithm returns two integers x,y, so gcd(p,q) = px + qy . The Extended Euclidean Algorithm is one of the basic algorithms used in cryptography. The algorithm is for example used for the RSA encryption. Furthermore, the algorithm is used in the Chinese Remainder Theroem. Further applications of the Extended Euclidean algorithm include key generation, prime number generator and the primality testing.

The algorithm was taken from the book [1] and adapted for BigInteger . The proof that it may well have a greatest common divisor of two integers p, q there, we read in [1]. For completeness, the greatest common divisor of 0 and 0 is set to 0, so is gcd(0,0) = 0. Thus, the greatest common divisor of two integers is never negative. We take from [1] the following theorem.

Theorem 1.1 1. If q = 0 then, the gcd(p,q) = |p|.
2. If b≠0 then, the gcd(p,q) = gcd(|q|,p mod |q|).

The algorithm is explained with an example.

Example We want to compute the gcd(123,23). According to Theorem 1.1 1.1 we obtain gcd(123,23) = gcd(23,123 mod 23) = gcd(23,8) = gcd(8,7) = gcd(7,1) = gcd(1,0) = 1.

First, check which of the two numbers is larger. The larger number is declared as a parameter one, the smaller one as parameter two. If both numbers are identical, no change is made, as specified in this case, gcd (p, p) = p have a clear. The function is called with gcd(p,q). Now, unless the number divided by the number p modulo q, and counted up the number q is zero. For the next iteration, the new p and q is the new q is the rest of the modul calculation q = p mod q. Since q is always the rest of the modul calculation and eventually this will be zero, will ensure that the algorithm terminates. We can compactly write the calculation in a table.

To obtain the values for X and Y to be calculated, we initialize the first two iterations follow:

x0 = 0, x1 = 1 and y0 = 1, y1 = 0.

With r0,…,rn+1 we signify the remainder sequence and with q1,…,qn the sequence of quotients that arise when applying the extended Euclidean Algorithm on p,q. The of construction of the two sequences (xk) and (yk) such that x = (-1)nxn und y = (-1)n+1yn is described by the following formulas.

xk+1 = qkxk + xk-1, yk+1 = qkyk + yk-1, 1 ≤ k ≤ n.
