Rules

Goal:

You win, if the sum of your numbers is greater than the sum of the Number Shark numbers.
Try to maximise your sum as much as you can.

Rules:

You always have the first turn.

Select an arbitrary number from the number selection area. The Number Shark eats any real divisors of this selected number, as long these divisors has not yet been eaten up before (e.g. 6 has the real divisors 1, 2, 3). Continue selecting of numbers, which have not been selected before or were eaten by the Number Shark in previous rounds. Pay attention, the Number Shark always needs a number. If you select a number without remaining free divisors, then the Number Shark will take this number and you will get nothing. The game is over with the selection of the last free number.

Description:

At the beginning, you must decide for a number set.

There are three possible ways to generate the number set (Preferences):

Number set generation mode 1: 1,...,maximum (maximum is the size of the number set).
Number set generation mode 2: minimum,...,maximum (maximum - minimum + 1 is the size of the number set).
Number set generation mode 3: n1, n2, ..., nk (ni is element of IN, 1<=i<=k, k is the size of the number set).

Furthermore you can select out of three support modes (Preferences):

Support mode 1: You play against the Number Shark according to the rules defined above without any support.
Support mode 2: Same as mode 1, but you will get an additional online support.
Support mode 3: Same as mode 1, but you will get a proposal for a solution in advance (button "Strategies+Solutions").

After pushing the button "New Game" you can start the game.

During the game, you can use the button "Undo" to correct only your last selection and the button "Eat" to accelerate the game by providing with numbers for the Number Shark.
