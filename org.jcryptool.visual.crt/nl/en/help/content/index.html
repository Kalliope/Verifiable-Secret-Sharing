Chinese Remainder Theorem

The demonstration is to show the functioning of the Chinese Remainder Theorem plug-in.

The Chinese Remainder Theorem plug-in can be started on the menu visualization and maximize his view.

The plug-in consists of two columns. On the left side the description and the computational procedure of the current step is presented and on the right side the input and results of the calculation. In theEquation group on the right side of the plug-in the parameters of the equation can be entered.

Positive numbers of any size can be used. By pressing the + button the amount of the equations can be increased and reduced by the - button. The minimum number of equations is two. After determining the number of equations and input parameters the next step can be entered by pressing the Step 2 button.

If the input parameters of the equations was not correct, a dialog box opens where the user can correct the entry.

The user has the ability to correct themselves or allow the entry by clicking the Suggestion button to propose a valid value from the program. In both cases the user have to log the enter with the Apply button. With the Ok button the user returns to the program and the dialog box closes. The group Step 2 is activated and the computational procedure is emphasized.

The user can now switch on by pressing the button Step 3 to the third step. It will appear in the group inverse the intermediate results of the calculations.

The accompanying explanation has been previously described in Step 2 group. Step 3 describes the calculation of the inverse element.

The next step is then shown in Step 4 in the Inverse group.

The calculation of the final result is described in the last step, Step 4.

The final result can now be displayed in the group Result by pressing the buttons Result.

For the user to verify the accuracy of the solution all the equations are shown in the Verify group again.

The user can display further solutions of the simultaneous congruences by clicking the Next and Previous buttons. The solutions are shown in the Result as well in the Verify group. To initiate a new calculation, it is enough if you click in the Equation group on the + or - button. It is performed by a reset of the input fields to the Equation group and you can add or remove other equations. This reset is possible from any step.

After passing through the algorithm, it is possible to export calculation. The plug-in supports the output as LaTex, PDF or CSV file.
